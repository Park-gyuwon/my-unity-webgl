학기 과제 1

1. 수업 중 만든 게임 서비스 URL
- https://park-gyuwon.github.io/my-unity-webgl/

2. 학기말 프로젝트 기획 내용

1) 게임의 제목 및 장르
게임 제목: 핀볼 바운스 (Pinball Bounce)
게임 장르: 2D 캐주얼 아케이드 게임


2) 게임 시놉시스
출발지에서 발사된 공이 스테이지 내의 다양한 장애물과 벽에 부딪히며 통통 튕겨 다니는 직관적인 아케이드 게임입니다. 
공이 벽이나 장애물에 부딪힐 때마다 점수가 올라가며, 플레이어는 공을 최대한 많이 튕겨서 고득점을 기록한 뒤 최종 도착지(골인 지점)에 공을 안전하게 도달시키는 것을 목표로 합니다.

3) 게임의 특징
유니티 2D 물리 엔진 활용: 유니티의 내장 물리 엔진(Rigidbody2D, PhysicsMaterial2D)을 활용하여 공이 벽에 부딪혀 튕기는 반사 작용을 복잡한 코딩 없이 자연스럽게 구현합니다.
충돌 누적 스코어 시스템: 공이 최종 도착지에 들어가기 전까지 벽에 부딪힌 횟수에 비례해 점수가 계속 쌓이는 시스템을 적용하여, 단순하지만 고득점을 노리는 재미를 제공합니다.
명확한 게임 구조와 UI: '출발 ➔ 장애물 충돌(점수 획득) ➔ 도착'으로 이어지는 누구나 이해하기 쉬운 직관적인 규칙을 바탕으로 하며,
 현재 점수와 성공 여부를 화면에 깔끔하게 보여주는 UI를 구성합니다.
